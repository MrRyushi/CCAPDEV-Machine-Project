This repository contains the main project for the course CCAPDEV in De La Salle University. This website is about a 
computer laboratory reservation system that allows students and/or a lab technician to reserve seats. The project is 
still in Phase 1 (MCO1) and only the front-end is developed. Note that the navigation bar should only show the links
to "View Rooms" and "Sign in" when the user has not yet logged in. And the navigation bar should only show the links 
to "View Rooms", "My Profile", and "Log out" when the user is signed in.
